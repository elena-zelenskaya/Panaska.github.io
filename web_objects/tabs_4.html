Phrasal verbs

Set off — to begin a journey.

The princess set off on a long journey.

Set sail — to begin a trip on a ship or boat.

They set sail on Aug. 3, 1492 from Spain.

Set foot — to enter, to step on.

Europeans set foot on North America many centuries ago.

Set up — to put together or arrange the parts of something before using it.

It took almost an hour to set up the tent in the rain.

Set to work — to begin doing something.

We have fixed the roof; now we will set to work on the kitchen.
